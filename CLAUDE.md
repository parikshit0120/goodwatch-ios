### 1.2 Product Philosophy

- **Two journeys exist:** "Pick For Me" (single recommendation) and "Explore" (browse/discover catalog)
- **"Pick For Me" is the core identity** — one movie at a time, no lists in this flow
- **Explore is the secondary journey** — full catalog with search, filters, tabs. Requires separate auth
- **Low regret over maximum delight** — safe recommendations
- **Availability is a hard gate** — never recommend something user can't watch
- **Animated/kids content hidden by default** until user matures (5+ interactions)

### 3.3 Recommendation Engine — How It Works

This is the most critical section. The engine lives in `GWRecommendationEngine.swift` and `GWSpec.swift`.

**Data flow:**
```
RootFlowView.fetchRecommendation()
    → SupabaseService.shared.fetchMovies()    (raw Movie array from Supabase)
    → GWMovie.from(movie)                     (converts to engine-compatible type)
    → GWNewUserContentFilter.shouldExclude()  (hides kids/animation for new users)
    → GWRecommendationEngine.recommend()      (THE core pipeline)
        1. Filter: validate each movie against 8 rules (language, platform, seen, runtime, content type, goodscore, tags, availability)
        2. Score: 4-weight formula → tag alignment (50%) + regret safety (25%) + platform bias (15%) + dimensional fit (10%)
        3. Select: softmax weighted random from top 10 (temperature=0.15)
    → Returns GWRecommendationOutput (single GWMovie? + optional GWStopCondition)
```

**CRITICAL: `RootFlowView.fetchRecommendation()` calls `SupabaseService.shared` directly → `engine.recommend()`. It does NOT go through `MovieRecommendationService`.**

### 3.4 Scoring Formula (INV-L02)

```
baseScore = (tagAlignment × 0.50) + (regretSafety × 0.25) + (platformBias × 0.15) + ((1 - dimensionalPenalty) × 0.10)
finalScore = baseScore + confidenceBoost   // max 5%, only after 10+ learned tags
```

- **Tag alignment**: weighted intersection of movie tags with user intent tags, weighted by learned TagWeightStore values
- **Regret safety**: safe_bet=1.0, acquired_taste=0.6, polarizing=0.4 (weighted by tag weights)
- **Platform bias**: accept/reject ratio per platform from history (neutral 0.5 if <3 interactions)
- **Dimensional penalty**: penalizes movies matching rejection patterns (too_long → long movies, not_in_mood → polarizing content)
- **Confidence boost**: 0-5% of tagAlignment, activated only when ≥10 tags deviate from default 1.0 (INV-L03)

### 3.5 GoodScore Priority (INV-D01)

```
GoodScore = composite_score > 0 ? composite_score : (imdb_rating ?? vote_average)
```

- `composite_score`: multi-source weighted score (0-100 scale) — only ~255/22,370 movies have this
- `imdb_rating`: IMDB score — 44% of catalog is null
- `vote_average`: TMDB rating — always available (0-10 scale, × 10 for threshold comparison)

### 3.6 Tag Weight Learning (INV-L01)

```
watch_now:       +0.15 per tag
completed:       +0.20 per tag
not_tonight:     -0.20 per tag
abandoned:       -0.40 per tag
show_me_another: -0.05 per tag
```

**TagWeightStore** lives inside `GWSpec.swift` (~line 524). Persists `[String: Double]` in UserDefaults, scoped by userId (INV-D03).

Default weight for any tag = 1.0. Deltas accumulate over time:
- After 2 rejections of "dark" movies: 1.0 → 0.6
- After 1 watch of "feel_good": 1.0 → 1.15
- After 5 skips of same tag: drops by 0.25

### 3.7 Maturity System (INV-R09)

Tracked in `InteractionService.swift` → `UserMaturityInfo`:
- `watchNowCount`: number of "Watch Now" taps
- `hasWatchedDocumentary`: boolean
- `isMatureUser`: true when `watchNowCount >= 5`

When `isMatureUser == false`:
- Animation/kids/family genres excluded (unless clearly adult-oriented: high rating + adult genre)
- Documentaries excluded
- Musicals excluded

### 3.8 Adaptive Quality Gate (INV-R06)

Two quality layers:
1. **GoodScore threshold** (mood-based, in `isValidMovie()`): Tired=88, Neutral=80, Adventurous=75
2. **Adaptive quality gate** (tier-based, in `recommend()`): Checks rating + vote count, auto-relaxes for small catalogs

The adaptive gate replaces all language-specific threshold overrides. It counts candidates after language+platform filtering and relaxes ONE step if < 10 candidates pass the strict gate. Absolute floor: 6.5 rating / 300 votes.

### 3.9 Not-Tonight Penalty (INV-L05)

When user rejects a movie, `recommendAfterNotTonight()`:
1. Adds rejected movie ID to exclusion set
2. Computes tag overlap between rejected movie and each candidate
3. Applies penalty: `overlapRatio × 0.3` subtracted from candidate score
4. Still uses weighted random from top 10 (prevents repetitive alternatives)

To commit changes to protected files:
```bash
./unlock.sh          # Creates 60-second unlock token
git commit -m "..."  # Must happen within 60 seconds
```

### 15.2 STOP-AND-ASK Checkpoint Rule

Before performing ANY of the following actions, **print what you are about to do and wait for explicit user confirmation**:

- `git commit` — state which files and the commit message
- `git push` — state which branch and remote
- `wrangler deploy` — state what is being deployed
- `rm` / `delete` / file removal — state which files
- HTML template changes — state which template and what changed
- SQL query changes — state which query and what changed
- RLS policy changes — state which table and policy
