# App Store Optimization (ASO) Skill

## Core Capabilities

**Research & Analysis:**
- Keyword research with volume, competition, and relevance scoring
- Competitor strategy analysis for top-performing apps
- Market trend identification and category evaluation
- Review sentiment analysis extracting user insights
- Rating improvement strategies

**Metadata Optimization:**
- Platform-specific title/description creation respecting character limits
- Apple subtitle (30 chars) and promotional text (170 chars) optimization
- Google Play short description (80 chars) handling
- Keyword field maximization (Apple's 100-char limit)
- Visual asset guidelines for icons and screenshots

**Conversion & Performance:**
- A/B testing framework for metadata and visuals
- Impression-to-install conversion optimization
- ASO health scoring (0-100 scale) across four categories
- Keyword ranking tracking and benchmarking
- Download velocity monitoring

**Launch & Updates:**
- Pre-launch compliance checklists for both platforms
- Optimal release timing recommendations
- Update cadence planning with feature rollouts
- Seasonal campaign identification
- Multi-language localization strategies

## Key Output Formats

The skill generates keyword research reports, optimized metadata packages, competitor analysis, ASO health scores with prioritized recommendations, A/B test plans with statistical significance calculations, and comprehensive launch checklists.

## Eight Supporting Scripts

Including `keyword_analyzer.py`, `metadata_optimizer.py`, `competitor_analyzer.py`, `aso_scorer.py`, `ab_test_planner.py`, `localization_helper.py`, `review_analyzer.py`, and `launch_checklist.py` providing modular functionality.
